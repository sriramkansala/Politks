# Product UI Anti-Patterns

Reject any output containing these tendencies unless the feature explicitly requires them.

## Visual Slop

- Purple or blue gradients added without brand justification
- Glassmorphism
- Excessive shadows
- Excessively rounded containers
- Floating cards nested inside floating cards
- Arbitrary accent colors
- Random illustrations inside serious product workflows
- Over-designed backgrounds
- Unnecessary blur effects
- Decorative animations
- Huge icons added to fill empty space

## Generic AI Dashboard Patterns

- Four generic KPI cards followed by a random chart
- Charts added without a real product reason
- Cards used for every section
- Oversized dashboard widgets
- Fake activity feeds added only to fill space
- Fake analytics added because the page looks empty
- Random badges or pills added to create visual interest
- A landing-page visual language applied to an operational interface

## Typography Slop

- Huge marketing-style headings inside product screens
- Excessive use of bold text
- Inconsistent font weights
- More than two unrelated font families
- Random monospace text used decoratively
- Poor line-height decisions
- Tiny unreadable supporting text
- All-caps labels used excessively
- Hierarchy created only through font size

## Interaction Slop

- Hover states that shift layouts
- Hidden primary actions
- Duplicate controls
- Unclear selected states
- Missing loading states
- Missing empty states
- Missing error states
- Missing disabled states
- Missing focus states
- Motion added only for spectacle
- Unnecessary confirmation dialogs
- Actions placed far away from the content they affect

## Copy Slop

- Generic text such as "Unlock your potential"
- Vague labels such as "Proceed" when a more specific action is known
- Excessive explanatory text
- Startup-marketing language inside operational screens
- Placeholder copy that does not represent realistic product content
- Repeated helper text that adds no value

## Final Rejection Question

Before approving any screen, ask:

"Does this look like a real product designed for repeated use, or like an AI-generated dashboard template?"

If the answer is uncertain, simplify and correct it.
